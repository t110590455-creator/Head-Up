# Head-Up
HeadUp is an Android posture protection app for K-12 students. Powered by MediaPipe Edge AI, it utilizes Cosine Foreshortening and Angular Velocity algorithms to preemptively warn users 2s before posture collapse. Features a 3s debounce mechanism, gamified UI, and local Room DB. Built with Kotlin &amp; CameraX.

### MediaPipe Samples
This repo hosts the official MediaPipe samples with a goal of showing the fundamental steps involved to create apps with our machine learning platform.
